# Project Title: HEART FAILURE SURVIVAL PREDICTION

Predicts heart failure patient survival using machine learning.
Uses medical data like age, blood pressure, and clinical parameters.
Trains models to analyze patient health data.
Identifies high-risk patients early.
Helps doctors make better treatment decisions.
